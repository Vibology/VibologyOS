# Archangel Entry Format Analysis

## Current Format Inconsistencies

### Three Distinct Formats Identified:

---

## FORMAT A: "Comprehensive Roman Numeral" (5 entries)
**Archangels:** Gabriel, Uriel, Kamael, Haniel, Raziel
**Line Count:** 441-776 lines
**Section Style:** Roman numerals (I-XIX)
**Section Count:** 15-19 sections

**Typical Structure:**
```
## I. Foundational Material
## II. Core Correspondences
## III. Qabalistic Attribution
## IV. Biblical Foundation
## V. Iconography and Symbolism
## VI. Function in Divine Economy
## VII. Human Consciousness Parallel
## VIII. The Tarot
## IX. Element
## X. Planetary Correspondence
## XI. Mythology and Cross-Tradition Parallels
## XII. Practical Mysticism
## XIII. Jungian and Psychological Dimension
## XIV. Shadow and Distortion
## XV. The Gift of Integration
## XVI. [Relationship to Other Archangels]
## XVII. Cross-System Correspondences
## XVIII. Keywords and Themes
## XIX. A Prayer to [Name]
## Internal Links
```

**Example:** Gabriel.md (753 lines, 19 sections)

---

## FORMAT B: "Comprehensive Non-Numbered" (5 entries)
**Archangels:** Michael, Raphael, Tzadkiel, Tzaphkiel, Metatron
**Line Count:** 540-1,064 lines
**Section Style:** Plain text headers (no Roman numerals)
**Section Count:** 15-19 sections

**Typical Structure:**
```
## Introduction: [Descriptive Title]
## Core Correspondences
## Hebrew Etymology and Mystical Significance
## Pseudo-Dionysius and Traditional Angelology
## Biblical and Scriptural Foundations
## Qabalistic Integration: [Name] and [Sephirah]

## Iconography and Symbolism
## Function in Divine Economy
## Human Consciousness Parallel

## Cross-Tradition Parallels and Mythological Resonances
## Jungian and Psychological Dimension
## Shadow and Distortion
## Practical Mysticism
## The Gift of Integration
## Cross-System Correspondences (sometimes)
## Internal Links
## Closing: [Descriptive Title] (sometimes, e.g., Michael)
```

**Example:** Michael.md (1,064 lines, 16 sections + Closing)

---

## FORMAT C: "Abbreviated" (1 entry)
**Archangel:** Sandalphon
**Line Count:** 473 lines
**Section Style:** Plain text headers
**Section Count:** 5 sections only

**Structure:**
```
## Foundational Material
## Core Correspondences
## Practical Application
## Synthesis Notes
## Internal Links
```

**Note:** This format appears incomplete compared to the others, lacking many standard sections (Tarot, Element, Mythology, Jungian analysis, Shadow/Gift, etc.)

---

## Key Differences

### 1. **Section Numbering**
- Format A: Roman numerals (I, II, III...)
- Format B: No numbering
- Format C: No numbering

### 2. **Opening Section Name**
- Format A: "Foundational Material" (Gabriel, Uriel, Kamael, Haniel, Raziel)
- Format B: "Introduction: [Descriptive Subtitle]" (Michael, Tzadkiel, Tzaphkiel) OR "Foundational Material" (Metatron, Raphael)

### 3. **Section Order Variations**
- Format A: Core Correspondences comes second
- Format B: Core Correspondences sometimes comes second, sometimes after Introduction
- Format B: Some entries include "Hebrew Etymology" as separate section early on
- Format B: Some entries include "Pseudo-Dionysius" section
- Format B: Some include "Steiner's Spiritual Hierarchies" section

### 4. **Depth Variations**
- Sandalphon lacks: Tarot, Element, Planetary, Mythology, Jungian, Shadow/Gift sections
- All other entries have 15-19 comprehensive sections

### 5. **Closing Variations**
- Gabriel (Format A): Ends with "A Prayer to Gabriel" before Internal Links
- Michael (Format B): Ends with "Closing: The Question and the Answer" after Internal Links
- Most others: End with Internal Links

---

## Recommended Uniform Format

Based on analysis, here are three options:

### OPTION 1: Roman Numeral Standard (Based on Gabriel)
- **Pros:** Clear hierarchical structure, easy to navigate, consistent numbering
- **Cons:** More formal, longer section titles

### OPTION 2: Descriptive Header Standard (Based on Michael)
- **Pros:** More evocative, allows for creative titles per archangel
- **Cons:** Harder to maintain consistency, no clear hierarchy

### OPTION 3: Hybrid Approach
- Use Roman numerals for core sections (I-XV)
- Allow optional creative Introduction/Closing outside numbering
- Standardize section names and order

---

## Observations

1. **Completeness varies significantly:** Sandalphon is substantially less comprehensive than the others (473 lines vs 753-1064 average)

2. **Gabriel and Michael are most comprehensive:** 753 and 1,064 lines respectively, serving as effective templates

3. **Newer entries use Format B:** Michael, Raphael, Tzadkiel, Tzaphkiel (higher line counts, more elaborate)

4. **Earlier entries use Format A:** Gabriel, Uriel, Kamael, Haniel, Raziel (Roman numerals)

5. **Possible chronology:** Format A entries written first, Format B refined later, Sandalphon abbreviated/incomplete

Use Gabriel.md as the **canonical template** with these adjustments:
